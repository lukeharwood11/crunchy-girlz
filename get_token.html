How to Use

  1. Configure your Supabase project details above
  2. Sign in using Google OAuth or email/password
  3. Copy the JWT token that appears
  4. Go to your FastAPI docs (http://localhost:8000/docs)
  5. Click the "Authorize" button
  6. Paste the token (without "Bearer " prefix)
  7. Now you can test protected endpoints!
